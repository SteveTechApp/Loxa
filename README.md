# Loxa Backend Server

This is the backend server for the Loxa messaging app.

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Run the server:
```bash
node server.js
```

3. Server will start on:
```
http://localhost:3000
```

Supports:
- Realtime messaging with Socket.IO
- Login and Signup endpoints
- Dummy translation API
